Introduction

The Tifinagh keyboard was developed to input characters for Tifinagh scripts. You can use this keyboard with any Tifinagh Unicode font.

For typing in the Tawallammat Tamajaq language, we highly recommend the Tagmukay font at http://software.sil.org/tagmukay. This font has the required characters and bi-consonant ligatures for Tawallammat.

The Tifinagh keyboard layout can be seen in the Tawallammat Tifinagh Keyboard Layout PDF document.
